# 이미지 특징 공간 분석 및 은닉층 차원 축소에 따른 효율성 검증 실험
> **본 프로젝트는 수업 내용을 바탕으로 진행한 개인적 호기심에서 출발하여, 이미지 데이터가 신경망 내부에서 다중 로지스틱 회귀 모형으로 전사되는 과정을 추적하고 은닉층의 차원 크기를 줄였을 때 분류 효율성이 어떻게 변화하는지 확인한 실험입니다.**

---

## 1. 연구 동기 및 개요 
학부 수업에서 일반적인 CNN 모델을 학습하면서 생긴 두 가지 의문은 다음과 같습니다.
    1) 모델이 어떤 연산 과정을 거치길래 복잡한 비정형 이미지를 최종단의 다중 로지스틱 회귀분석 스페이스로 정렬해내는지
    2) 신경망의 구조와 각 층의 은닉 차원의 크기를 결정하는 명확하고 정량적인 기준이 존재하는가

일반적으로 합성곱 레이어들은 비선형 변환을 반복하며 뒤엉킨 이미지 픽셀 데이터들을 최종 분류 직전의 잠재 공간 상에 클래스별로 무리를 지어 수렴시키는 공간 정렬 역할을 수행합니다. 이때 학습이 잘된 모형일수록 각 사물의 특징 벡터들이 서로 멀리 떨어져 명확한 군집을 이루게 됩니다.

여기서 한 가지 의문이 생겼습니다. 앞단 특징 추출기 구간에서 사물의 특징을 충분히 잘 모아놓았다면, 굳이 최종 분류기 직전의 공간을 넓게 쓸 필요가 없이 과감하게 줄여도 되지 않을까 하는 점이었습니다. 정보 정렬을 전담하는 특징 추출기의 차원(채널 수)을 줄이는 실험도 고려했으나 이 경우에는 이미지에서 특징을 포착하는 해상도가 무너져 분석이 비교적 쉽지 않아 보류했습니다. 

따라서 본 실험에서는 특징 추출을 담당하는 전반부 특징 추출기는 동일하게 통제하여 고차원 특징 벡터(16,384차원)의 구조로 고정했습니다. 또한 모인 정보를 전달받는 분류 직전의 Squeeze Layer의 차원 크기만을 64차원에서 8차원으로 87.5% 압축하는 대조 실험을 설계했습니다. 

학습 완료 후, 이 Squeeze Layer에서 추출한 특징 벡터들을 t-SNE 알고리즘을 통해 2차원 평면에 사영함으로써, 앞단 CNN이 정렬해 놓은 데이터 분포가 초저차원 공간으로 압축될 때도 클래스 간의 선형 분리성을 강건하게 유지하는지 확인하고자 했습니다.

## 2. 사용 데이터셋 
모형 내부의 특징 공간 분포 변화를 명확하게 추적하기 위해, 사물의 형태와 배경 노이즈가 복잡한 CIFAR-10 표준 사물 이미지 데이터셋을 채택하여 전이 학습 없이 직접 모형을 학습해 진행합니다.

* **데이터 규모:** 총 60,000장 (32 × 32 해상도, RGB 3채널 컬러 이미지)
  * **Train Set:** 50,000장 (클래스당 5,000장 배정)
  * **Test Set:** 10,000장 (학습 루프와 완전히 분리하여 평가의 독립성 확보)
* **전처리 파이프라인:**
  * 이미지 픽셀 텐서화 및 $[0, 1]$ 실수 스케일링 수행.
  * RGB 채널별 Z-Score 표준화($\mu=0.5, \sigma=0.5$)를 적용하여 입력 분포를 $[-1.0, +1.0]$ 공간으로 정렬함.

링크: https://cave.cs.toronto.edu/kriz/cifar.html

## 3. 실험 설계 및 변수 통제 
특징 추출 능력의 변화에 따른 성능 교란 현상을 방지하기 위해, 사물의 시각적 특징을 추출하는 전반부 아키텍처는 고정하고 최후단 은닉 차원의 용량만 조절하는 변수 통제 실험을 수행합니다.

* **통제 변수:** 전반부 4층 구조의 Dense-CNN 특징 추출기. 모든 실험군은 입력 이미지로부터 일관되게 가공된 16,384차원의 고차원 특징 벡터를 동일하게 공급받습니다.
* **조작 변수:** 분류기 직전 은닉층 차원의 크기.
  * **실험군 A (기준 용량):** $64\text{차원}$ (잠재 공간의 분산을 충분히 수용)
  * **실험군 B (압축 용량):** $8\text{차원}$ (정보 표현 공간의 크기를 87.5% 제한)
* **Baseline:** 학부 수업 표준 예제 규격의 일반 바닐라 CNN (채널 $8 \to 16$ 레이어 적층 모형)을 대조군으로 배치하여, 본 실험에서 채택한 Dense-Block 구조가 특징 정렬 및 수렴 속도 측면에서 발휘하는 효율성을 비교 평가합니다.

## 4. 모델 아키텍처 및 수리적 성질 
국소적 특징과 추상적 전역 맥락을 동시에 보존하며 은닉층으로 전달할 수 있도록 수제 설계한 Dense CNN 구조입니다.

[Input: 3x32x32]  
│  
[Conv 1] ➔ 국소적 경계 특징 추출 (h1) ──────┐  
│                                      │ (Dense Connection: 채널 축 결합)  
[Conv 2] ➔ 추상적 공간 맥락 추출 (h2) ──────┴─➔ [dense_bridge: 64x32x32]  
│  
[Conv 3 + MaxPool] ➔ 차원 수축 및 공간 불변성 확보 [128x16x16]  
│  
[Conv 4 + MaxPool] ➔ 고차원 특징 스케일업 [256x8x8]  
│  
[Flatten] ➔ 고차원 특징 전사 벡터 생성 [16,384차원]  
│  
🌟[FC Bottleneck] ➔ 은닉층 차원 제어 및 실험 구간 (64차원 vs 8차원 선형 사영)  
│  
[FC Classifier] ➔ 다중 로지스틱 분류 경계 형성 [10차원 로짓 배출]

## 5. 실증 결과 비교 및 고찰 

### (1) 정량적 성능 지표 비교
학습 루프와 완전히 격리된 10,000장의 검증 데이터셋(Test Set)을 투입하여 최종 분류 정확도를 산출한 결과는 다음과 같습니다.

| 평가 지표 (Metric) | Baseline (학과 수업 표준 예제) | 실험군 A (64차원 은닉층) | 실험군 B (8차원 은닉층) |
| :--- | :---: | :---: | :---: |
| **최종 검증 정확도 (Test Acc)** | $58\% \sim 65\%$ | **$75.37\%$** | **$69.22\%$** |
| **공간 자원 절약률 (은닉층)** | 기준점 | 용량 최적화 | **$87.5\%$ 공간 압축** |
| **성능 변화 폭 (vs 64차원)** | - | 기준점 | $\Delta$ **$-6.15\%p$** |

* **결과 해석:** 최종 은닉층의 차원 크기를 64차원세서 8차원으로 **87.5%** 축소시켰으나 최종 검증 정확도의 하락 폭은 **6.15%p** 수준으로 방어해냈습니다.

### (2) t-SNE 기반 잠재 공간 정성 분석
분류층 직전 은닉층에서 추출한 특징 벡터들의 데이터 분포를 t-SNE 알고리즘으로 시각화하여 내부 특징 공간의 물리적 실체를 대조 확인했습니다.

#### 실험군 A : 64차원 은닉층 (`./manifold_report_dim_64.png`)
* **밀도 차이로 보이는 경계:**
  전체적으로 사물들은 동물 범주와 교통수단 범주라는 두 개의 구역으로 양분되어 모여 있습니다. 특히 두 대분류 구역 사이에는 데이터 점들이 비교적 찍히지 않은 밀도가 낮은 구역이 존재하며, 이는 모형이 두 도메인 간의 시각적 이질성을 통계적으로 인지하고 경계면을 형성했음을 보입니다.
* **시각적 맥락 공유에 따른 매니폴드 연장:**
  가장 주목할 만한 현상은 새(bird) 클래스의 분포입니다. 새 군집은 동물 도메인 내부에 넓게 퍼져 있으나, 특이하게도 교통수단 도메인의 비행기 구역까지 퍼져 있는 형태를 보입니다. 이는 모형이 단순 정답지 라벨을 외운 것이 아니라, '날개', '공중을 나는 구도', '하늘 배경'등 두 클래스 간의 시각적 유사성을 잠재 공간에 나타냈음을 보입니다.

#### 실험군 B : 8차원 은닉층 (`./manifold_report_dim_8.png`)
* **데이터 정렬 양상:** 이동수단 클래스들은 여전히 선형 경계면을 유지하며 독립적으로 군집을 이루고 있습니다. 중앙의 동물 클래스들 역시 다소 중첩이 있으나 여전히 구별력을 유지합니다. 
* 87.5%의 차원을 축소했으나 특징 공간의 표현력이 유지됨을 확인했습니다.

## 6. 결론 
본 실험을 통해 두 가지 학술적 사실을 도출했습니다.

1. **다항회귀 전사 과정의 시각적 확인:** 픽셀 데이터들이 전반부 Dense CNN 구조를 통과하면서 잠재 공간 상에 클래스별로 군집을 이루며 정렬된다는 사실을 t-SNE를 거쳐 산점도를 통해 시각적으로 확인했습니다. 
2. **은닉층 차원 결정의 효율성 비교:** 은닉층 공간을 무작정 넓게 쓰는 것이 무조건적인 성능 향상을 보장하지 않으며, 상당 부분 공간적 낭비가 존재하고 있었음을 확인했습니다. 차원을 8차원까지 수축시켜도 특징 공간의 선형 분리성이 강건하게 유지된다는 것은, 앞단 Dense CNN이 사물의 본질적인 핵심 특징만을 매우 높은 밀도로 압축하여 전달하고 있음을 의미한다고 해석했습니다.
